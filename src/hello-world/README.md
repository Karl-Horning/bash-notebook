# `hello-world`

A Bash script that prints "hello world" to the terminal.

## What it does

```bash
echo hello world
```

## Usage

Make the script executable:

```bash
chmod +x hello-world.sh
```

Run it:

```bash
./hello-world.sh
```

Output:

```text
hello world
```

## Why this exists

The traditional starting point for learning Bash. It confirms:

- Your script runs
- Permissions are set correctly
- Your terminal works
